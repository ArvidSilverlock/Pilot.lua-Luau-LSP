  • Hull

Hull

A legacy pre-component part. Is able to float in the water, as well as carry objects attached to it. Primarily used for creating boats and other water craft.

It is a spawnable non-flammable solid.

It cannot be resized.

At its default size (16x4x6) it has a durability of 1.

By default, its colour is #dfdfde.

It requires 1 Stone, to be crafted.

Events

ComponentsUpdated(...)

This event never seems to fire, only tested on the deprecated Blade part, though.

The variadic parameter is unknown. It can be anything.
